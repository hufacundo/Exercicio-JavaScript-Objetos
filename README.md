# Exercicio-JavaScript-Objetos
Crie dois objetos com a estrutura abaixo:  - Nome (tipo String) - Idade (tipo Number) - Filiacao (Array): nomeDoPai (tipo String) e nomeDaMae (tipo String) - Endereco (tipo Objeto): rua (tipo String), cidade (tipo String) e estado (tipo String) - dadosCompletos (tipo função) (verifica se os dados acimas estão preenchidos)   Crie uma função que receba os dois objetos acima criados, imprima os valores de suas propriedades e por fim imprima o resultado da função chamada dadosCompleto() que deve verificar se todos os campos estão preenhidos.
